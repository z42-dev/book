# 导读

Z42希望为搭建垂直社区一个通用的开源技术解决方案。

当前仍处于开发阶段。等第一版发布后，将作为 [tech2ipo.com](http://tech2ipo.com) 的新版上线。

待网站开发完成后，将继续开发手机版本。
